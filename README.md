# WPF Loaders

A collection of animated loaders to use in WPF projects.
You know, for fun, 'cause I'm bored sometimes.

**Available on [NuGet](https://www.nuget.org/packages/MrMitch.Loaders/) !**

```ps1
PM> Install-Package MrMitch.Loaders
```
